Juego propio: Tragamonedas (slot)

El juego consiste en detener la ruleta de imagenes, si se presta atencion se puede ganar facilmente. Para ganar se debe obtener 3 coincidencias. El juego nunca termina, puedes girar cuantas veces quieras. Proximamente: apostar x cantidad de dinero y de acuerdo a las coincidencias se obtiene el doble o triple de dinero apostado.
